# Ebola Outbreak Emergency Tracker Dashboard

## Project Overview

This project is an interactive Power BI dashboard developed to support public health surveillance and outbreak response during an Ebola outbreak. The dashboard tracks confirmed cases, fatality rates, healthcare system strain, and patient isolation patterns to provide timely insights for monitoring and emergency decision-making.

## Dashboard Objectives

* Monitor the progression of an Ebola outbreak over time
* Track key outbreak and healthcare response indicators
* Assess healthcare system strain and bed occupancy
* Support public health response through visual outbreak monitoring

* Country: Uganda

## Key Metrics

* Total Confirmed Cases:600
* Case Fatality Rate: 50%
* Bed Occupancy Rate: 1253%
* Healthcare Worker Infection Rate: 20%

## Dashboard Analysis Sections

* Total Confirmed Cases by Month and Isolation Status – monitors outbreak trends over time
* Bed Capacity Strain – tracks pressure on treatment and isolation centers
* Outbreak Demographic Matrix – summarizes outbreak patterns by population characteristics
